# OpenTT
Open Test Taker - Interactive Python testing suite

###### Example usage:

*No limit; -l N where N <= 0*
python main.py -f example.json [-l 0]

*Limit; -l N where N > 0*
python main.py -f example.json -l 5

*Random questions*
python main.py -f example.json -r

*Shuffle answers*
python main.py -f example.json -s
